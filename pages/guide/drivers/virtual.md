::: en
A virtual storage driver that generates random files and folders for testing and demonstration purposes.
:::
::: zh-CN
一个虚拟存储驱动，可生成随机文件和文件夹，用于测试和演示目的。
:::

## Number of files { lang="en" }

## 文件数量 { lang="zh-CN" }

::: en
The number of random files to generate.
Default: 30
:::
::: zh-CN
要生成的随机文件数量。
默认值: 30
:::

## Number of folders { lang="en" }

## 文件夹数量 { lang="zh-CN" }

::: en
The number of random folders to generate.
Default: 30
:::
::: zh-CN
要生成的随机文件夹数量。
默认值: 30
:::

## Minimum file size (bytes) { lang="en" }

## 最小文件大小 (字节) { lang="zh-CN" }

::: en
The minimum size of generated files.
Default: 1048576 (1MB)
:::
::: zh-CN
生成文件的最小大小。
默认值: 1048576 (1MB)
:::

## Maximum file size (bytes) { lang="en" }

## 最大文件大小 (字节) { lang="zh-CN" }

::: en
The maximum size of generated files.
Default: 1073741824 (1GB)
:::
::: zh-CN
生成文件的最大大小。
默认值: 1073741824 (1GB)
:::

## Download functionality { lang="en" }

## 下载功能 { lang="zh-CN" }

::: en
This virtual storage does not support actual file downloads as it only generates simulated files.
:::
::: zh-CN
此虚拟存储不支持实际文件下载，仅生成模拟文件。
:::
